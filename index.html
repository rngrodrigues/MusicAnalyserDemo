Sobre o Projeto

O MusicAnalyser é uma aplicação desktop desenvolvida em Python para análise eficiente de grandes datasets musicais. Ideal para estudos, experimentos e visualização de rankings por gênero.

Importa CSV, Excel, JSON e Parquet
Processamento rápido com DuckDB
Top 10 interativo por gênero
Exporta PNG, PDF, CSV e Excel

Tecnologias Utilizadas

Python Pandas DuckDB Matplotlib Tkinter
